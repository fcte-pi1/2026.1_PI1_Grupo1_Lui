# Energia

A equipe de energia tem como objetivo garantir que o micromouse tenha um sistema de alimentação elétrica confiável, capaz de sustentar toda a operação do robô durante a prova, sem falhas ou perda de desempenho.

## Atuação

Durante o desenvolvimento do projeto, serão realizadas as seguintes atividades:

1. Entender a demanda energética do sistema
2. Modelar e estimar o consumo total
3. Dimensionar corretamente a fonte
4. Projetar o sistema de alimentação
5. Estimar e validar a autonomia
6. Validar com testes reais
