# Revisão e Merge

## Processo

1. Abrir o PR
2. Aguardar revisão de outro membro da equipe
3. Fazer ajustes se houver comentários
4. Merge realizado após aprovação

---

## ⚠️ Regras

- Não fazer merge sem revisão de pelo menos um membro
- Responder todos os comentários antes do merge
- Manter comunicação ativa com a equipe

---

## 🎯 Objetivo

Garantir qualidade, evitar erros e manter o projeto estável para todos.
