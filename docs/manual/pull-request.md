# Pull Request (PR)

## O que é?

Um Pull Request é um pedido formal para integrar as mudanças de uma branch em outra. É também o momento de **revisão** do trabalho.

---

## Como criar

1. Ir ao GitHub
2. Clicar em **"Compare & pull request"** (botão verde que aparece após o push)
3. Configurar:
    - **Base:** `engenharia`
    - **Compare:** sua branch
4. Escrever título e descrição
5. Clicar em **"Create pull request"**

---

## 📝 Descrição padrão recomendada

```markdown
Resolve #25

Implementa a funcionalidade X.

Inclui:
- Item 1
- Item 2
```

---

## Boas práticas

- Título claro e direto
- Descrição objetiva
- Sempre referenciar a issue relacionada
- Não abrir PR sem ter testado localmente
